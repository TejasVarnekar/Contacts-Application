# My Contacts Manager

A simple web-based Contacts Manager built with **Flask** and **SQLite**. This project allows users to **add, search, sort, and delete contacts**. Itâ€™s designed as a beginner-friendly project for learning Flask and SQLAlchemy.

---

## Features

- Add new contacts with **Name, Email, and Phone number**  
- Search contacts by name  
- Sort contacts by name (ascending/descending)  
- Delete contacts  
- Input validation for phone numbers (only 10-digit numbers allowed)  
- Responsive UI using **Bootstrap 5**
